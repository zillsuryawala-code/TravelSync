# TravelSync 🌍

A full-stack travel recommendation and hotel booking platform developed using Flask and MySQL.

## 📖 Overview

TravelSync helps users discover travel destinations and book hotels based on their preferences such as city, budget, and trip duration. The platform provides personalized recommendations, secure booking functionality, and an interactive chatbot for user assistance.

## ✨ Features

- Personalized travel recommendations
- Hotel booking system
- OTP-based email verification
- Chatbot integration
- User authentication and registration
- Feedback collection system
- Responsive user interface
- MySQL database integration
- Dark mode support

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/zillsuryawala-code/TravelSync.git
```

2. Install required packages

```bash
pip install flask
```

3. Configure MySQL database

4. Run the application

```bash
python app.py
```
